💡 Backup tips

  • Save JSON backups to a cloud folder (Drive / Dropbox / OneDrive)
  • Export CSV monthly for tracking progress in spreadsheets
  • Markdown reports are perfect for annual performance reviews
  • Browser data can be cleared accidentally — back up weekly

📅 Auto-snapshots (last 7 days)

Created automatically each day. Restore brings back that day's full state; the current data is auto-snapshotted again before restore so you can roll back.
